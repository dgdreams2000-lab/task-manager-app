# Task Manager App

A starter full-stack task manager application with an Express backend and a vanilla JavaScript frontend.

## Features
- Create tasks
- Toggle completed status
- Delete tasks
- REST API
- Simple responsive UI

## Tech Stack
- Node.js
- Express
- Vanilla JavaScript
- HTML/CSS

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   npm start
   ```
3. Open `http://localhost:3000`

## API Endpoints
- `GET /api/tasks`
- `POST /api/tasks`
- `PATCH /api/tasks/:id`
- `DELETE /api/tasks/:id`
